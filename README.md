# KEY SETTING

- 搜索
    - GET /api/keys?name=${name}
- 增
    - POST /api/keys
        - name: string; 平台名称
        - key: string;
        - secret: string;
        - clientID: string;
        - scope: string;
- 删
    - DELETE /api/keys
        - id: ObjectId;
- 改
    - PUT /api/keys
        - id: ObjectId;
        - key: string;
        - secret: string;
        - clientID: string;
        - scope: string;
- 查 根据ID查询单条记录
    - GET /api/keys/:id
